# ReallyWorld — сайт сервера

Статический сайт (переработан из шаблона форума под тематику Minecraft-сервера ReallyWorld), готовый к публикации на GitHub Pages.

## Структура

```
index.html   — вся разметка и стили (стили внутри <style>, без внешних CSS-фреймворков)
script.js    — мобильное меню, копирование IP, счётчик онлайна, скролл-анимации
```

Внешние зависимости — только шрифты Google Fonts (Press Start 2P, JetBrains Mono, Manrope), подключённые через `<link>`. Никаких сторонних JS-библиотек не требуется.

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub (например, `reallyworld-site`).
2. Загрузите `index.html` и `script.js` в корень репозитория (через веб-интерфейс "Add file → Upload files" или `git push`).
3. В репозитории откройте **Settings → Pages**.
4. В разделе **Source** выберите ветку `main` и папку `/root`, нажмите **Save**.
5. Через минуту сайт будет доступен по адресу `https://<ваш-логин>.github.io/reallyworld-site/`.

Если хотите использовать свой домен — добавьте файл `CNAME` с доменом внутри и настройте DNS согласно [документации GitHub Pages](https://docs.github.com/pages/configuring-a-custom-domain-for-your-github-pages-site).

## Что стоит донастроить перед публикацией

- **Счётчик онлайна** — сейчас это заглушка (`— / 9000`) в `script.js`. Чтобы показывать реальный онлайн, подключите свой backend/API, который опрашивает `mc.reallyworld.ru` по протоколу Server List Ping, и обновите значение `onlineNum.textContent`.
- **Ссылки на привилегии** — карточки в разделе «Магазин» ведут на общий раздел `#privileges` и на страницу `reallyworld.ru/description`. Если у вас есть прямые ссылки на покупку каждой привилегии, добавьте их на карточки.
- **Цены** — сайт сознательно не указывает конкретные цены привилегий (на исходном сайте цена определяется в личном кабинете/магазине). При необходимости добавьте блок `<div class="tier-price">` с реальной ценой.
- **Реквизиты в подвале** — взяты с открытой страницы reallyworld.ru; проверьте актуальность перед публикацией.

## Дизайн

Тема сборки — "инвентарь Minecraft": карточки привилегий стилизованы под слоты инвентаря (пиксельная рамка, врезанная тень), пиксельный шрифт `Press Start 2P` используется точечно (лого, заголовок, цифры статуса), а `JetBrains Mono` — для команд (`/kit hero`, IP-адреса), чтобы визуально отличать игровые команды от обычного текста.
